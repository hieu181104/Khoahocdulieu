## Hướng dẫn chạy code
- Tải xuống file `.ipynb`và file dữ liệu `.csv`
- Lưu trong cùng một thư mục
- Mở thư mục trong IDE (ví dụ Visual Studio Code)
- Chọn môi trường cho file `ipynb` thực thi:
```
Python 3.10
```
- Môi trường này cần cài các thư viện sau: pandas, matplotlib, numpy, seaborn, jupyter notebook.
```
pip install jupyter pandas numpy matplotlib seaborn
```
- Chạy từng khối lệnh và quan sát kết quả thực thi
